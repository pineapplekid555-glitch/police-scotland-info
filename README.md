# Police Scotland Information Hub

A comprehensive website providing knowledge and information about Police Scotland operations, state codes, procedures, and reference materials.

## Features

### 📋 State Codes
- Comprehensive database of Police Scotland state codes
- Crime classification codes (01-08)
- Drug-related codes (11-15)
- Traffic and vehicle codes (21-25)
- Administrative codes (31-35)
- Search functionality to find codes by code number or description
- Organized by category with severity indicators

### ⚙️ Procedures
- Community Policing Procedures
- Evidence Collection and Handling
- Stop and Search Procedures
- Arrest and Custody Procedures
- Incident Report Writing
- Mental Health Crisis Response

### 📚 References
- Official Police Scotland resources and links
- Legal framework and legislation
- Contact information (Emergency and non-emergency)
- Support resources for victims and those in need
- Key definitions and terminology

## Code Categories

### Severity Levels
- **Serious** (Red Badge) - Major crimes requiring immediate investigation
- **Moderate** (Yellow Badge) - Medium-level offences
- **Minor** (Cyan Badge) - Low-level violations
- **Administrative** (Green Badge) - Non-crime police activities

## Usage

1. **Browsing**: Navigate through different sections using the top navigation menu
2. **Searching Codes**: Use the search box on the State Codes page
3. **Learning Procedures**: Read through detailed procedures on the Procedures page
4. **Finding Resources**: Access official links and contact information on the References page

## Legal Information

Police Scotland operates under:
- Police and Fire Reform (Scotland) Act 2012
- Criminal Procedure (Scotland) Act 1995
- Human Rights Act 1998
